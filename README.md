# Phrase Bot

Телеграм-бот для игры с фразами из двух слов.

## Что изменено
- токен вынесен в `.env`
- база фраз хранится в `app/data/phrases.json`
- Excel в рантайме не используется
- при запуске бот проверяет и обновляет список фраз
- потом делает повторную проверку раз в сутки
- в базе остаются только фразы:
  - ровно из двух слов
  - каждое слово минимум 4 буквы
  - каждое слово минимум 2 слога

## Запуск

```bash
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
# впиши токен в .env
python main.py
```

## Структура
- `main.py` — точка входа
- `app/config.py` — настройки
- `app/handlers/` — хендлеры
- `app/services/` — работа с фразами, картинкой и обновлением базы
- `app/data/phrases.json` — локальная база фраз
- `app/data/update_state.json` — служебное состояние обновления
- `app/assets/fonts/` — шрифты

## Источник обновлений
Скрипт подтягивает открытый список фразеологизмов из Русского Викисловаря и объединяет его с локальной базой.

Атрибуцию к открытому источнику см. в `app/data/sources.json`.
